=========================
Function: isEmpty()
Precondition: Priority Queue is initialized
Postcondition: Returns True if queue is empty, else False
Time Complexity: O(1)
Edge Cases:
- Queue is empty initially (size == 0)
- Queue becomes empty after dequeuing all elements
=========================
Function isEmpty():
    if size == 0:
        return True
    else:
        return False

=========================
Function: isFull()
Precondition: Priority Queue is initialized
Postcondition: Returns True if queue is full, else False
Time Complexity: O(1)
Edge Cases:
- Queue is full when size == max_size
=========================
Function isFull():
    if size == max_size:
        return True
    else:
        return False

=========================
Function: enqueue(value, priority)
Precondition: Priority Queue is initialized
Postcondition: Inserts element with given priority if not full
Time Complexity: O(1)
Edge Cases:
- Queue overflow: cannot enqueue if queue is full
- First element insertion (size == 0)
- Negative or zero priority (optional)
=========================
Function enqueue(value, priority):
    if isFull():
        print("Queue Overflow")
    else:
        data[size] = (value, priority)
        size = size + 1
        print("Enqueued:", value, "with priority:", priority)

=========================
Function: dequeue()
Precondition: Priority Queue is initialized
Postcondition: Removes and returns element with highest priority if not empty
Time Complexity: O(n)
Edge Cases:
- Queue underflow: cannot dequeue if queue is empty
- Multiple elements with same highest priority: removes first one encountered
- Last element in the queue: size becomes 0 after removal
=========================
Function dequeue():
    if isEmpty():
        print("Queue Underflow")
        return None
    else:
        max_index = 0
        for i from 1 to size - 1:
            if data[i].priority > data[max_index].priority:
                max_index = i
        item = data[max_index].value
        for i from max_index to size - 2:
            data[i] = data[i + 1]
        size = size - 1
        print("Dequeued:", item)
        return item

=========================
Function: display()
Precondition: Priority Queue is initialized
Postcondition: Prints all elements with their priorities
Time Complexity: O(n)
Edge Cases:
- Queue is empty: prints “Priority Queue is empty”
- Single element queue: prints that element only
=========================
Function display():
    if isEmpty():
        print("Priority Queue is empty")
    else:
        for i from 0 to size - 1:
            print("Value:", data[i].value, "Priority:", data[i].priority)
